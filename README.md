### 프로젝트 멤버
- 이효원 : 팀장, 프로모션, 구매, 포인트관리
- <b>문덕용</b> : 게시판, 고객센터, 관리자
- 안시진 : 로그인/회원가입
- 김동욱 : 홈페이지 메인, 놀이공원 목록, 위시리스트
- 윤일준 : 놀이공원 및 놀이기구 상세정보, 실시간 채팅

### 담당 기능
- 커뮤니티 게시판(board)
- 고객센터(customerservice)
  - 공지사항
  - FAQ
  - 1:1 문의게시판
- 관리자페이지(admin)
  - 회원 권한 설정
  - 놀이공원 추가
  - 놀이기구 추가
<br/>



### 사용기술
- RestAPI를 이용한 페이지 이동없이 비동기적 로딩
- commons-fileupload를 사용한 파일 업로드
- mybatis를 이용한 DB통신
